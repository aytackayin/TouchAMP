Cron Schedule Samples & Scenarios
Schedule Format Guide
Schedule Description
* * * * * Every minute
*/5 * * * * Every 5 minutes
0 * * * * Every hour (at minute 0)
0 0 * * * Every day at midnight (00:00)
0 0 * * 0 Every Sunday at midnight
0 0 1 * * First day of every month at 00:00
0 0 1 1 * Every New Year's Day (Jan 1st)
Practical Scenarios
1. Triggering a Website URL
Useful for triggering PHP cron jobs on remote servers or local sites.
curl -s "http://example.com/cron.php"
2. Running a Local Node.js Script
Run a script located in your TouchAMP root or project folder.
node test-cron.js
3. Running a Local PHP Script (CLI)
Execute PHP files directly using the built-in PHP binary.
php www/my-project/cron.php

Apache Installation Guide

To download and install a compatible Apache version, follow these steps:

  • Go to the Apache Lounge download page.
  • Download the latest Apache Win64 (x64) .zip file.
  • Select the downloaded .zip file using the Manual Install button.
  • The app will automatically install it to the correct folder.
PHP Installation Guide

To download and install a compatible PHP version, follow these steps:

  • Go to the official PHP Windows download page.
  • Make sure to download the Thread Safe x64 version (.zip) for Apache compatibility.
  • Select the downloaded .zip file using the Manual Install button on this page.
  • The app will automatically place the files and configure the settings.
MySQL Installation Guide

To download and install a compatible MySQL version, follow these steps:

  • Go to the official MySQL Community download page.
  • Always download the ZIP Archive version (not the installer).
  • Select the downloaded .zip file using the Manual Install button on this page.
  • The app will automatically create database folders and configure the settings.
